# Windows 빌드 최종 가이드

## 문제 원인

electron-builder가 빌드 시 `winCodeSign` 도구를 다운로드하려고 합니다.
이 도구 안에 macOS용 심볼릭 링크가 있어서 Windows에서 추출 시 권한 오류가 발생합니다.

## 해결 방법 (확실함)

### 방법 1: 관리자 권한으로 실행 (가장 확실함) ⭐

1. **명령 프롬프트를 관리자 권한으로 실행**
   - Windows 검색에서 "cmd" 검색
   - "명령 프롬프트" 우클릭 → "관리자 권한으로 실행"

2. **다음 명령어 실행:**
```cmd
cd C:\Users\ASUS\Downloads\video-converter-source
npm install
npm run build:win
```

**관리자 권한이면 심볼릭 링크 생성이 가능하므로 오류가 발생하지 않습니다.**

---

### 방법 2: Windows 개발자 모드 활성화

1. **Windows 설정 열기**
   - Windows 키 + I

2. **개발자 모드 활성화**
   - 설정 → 개인 정보 보호 및 보안 → 개발자용
   - "개발자 모드" 켜기

3. **빌드 실행:**
```cmd
cd C:\Users\ASUS\Downloads\video-converter-source
npm install
npm run build:win
```

**개발자 모드가 활성화되면 일반 사용자도 심볼릭 링크를 생성할 수 있습니다.**

---

### 방법 3: 수동 캐시 준비 (복잡함)

1. `winCodeSign-2.6.0.7z` 다운로드
2. 관리자 권한으로 압축 해제
3. 캐시 폴더에 복사

**이 방법은 복잡하므로 방법 1을 권장합니다.**

---

## 권장 순서

1. **먼저 방법 1 시도** (관리자 권한)
2. 안 되면 **방법 2 시도** (개발자 모드)
3. 그래도 안 되면 방법 3

---

## 참고

이 문제는 electron-builder의 알려진 이슈입니다.
설정만으로는 해결되지 않으며, Windows 권한 설정이 필요합니다.

**가장 간단하고 확실한 방법은 관리자 권한으로 빌드하는 것입니다.**
