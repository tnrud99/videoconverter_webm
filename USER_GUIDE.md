# VirtueEdit Video Converter - User Guide

## Quick Start

1. **Download** the `VirtueEdit Video Converter.exe` file
2. **Double-click** to launch the application
3. **Select** a WebM file (drag & drop or click "Select File")
4. **Choose** your preferred quality settings (optional)
5. **Click** "Start Conversion"
6. **Wait** for the conversion to complete
7. The converted MP4 file will be saved in the same folder as your original file

## System Requirements

- **Windows 10 or later**
- No installation required (portable application)

## Features

- ✅ Convert WebM videos to MP4 format
- ✅ Adjustable video quality (CRF 18-26)
- ✅ Multiple resolution options (360p to 2160p)
- ✅ Audio bitrate options (128-320 kbps)
- ✅ Real-time progress tracking
- ✅ No installation needed

## Supported Formats

- **Input**: WebM files only (`.webm`)
- **Output**: MP4 files (`.mp4`)

## Quality Settings

### Video Quality
- **Original Plus** (CRF 18) - Best quality, slower conversion
- **High-Quality Web** (CRF 20) - Recommended for web use
- **Balanced** (CRF 23) - Good balance of quality and file size
- **Lightweight** (CRF 26) - Smaller files, faster conversion

### Resolution Options
- **2160p** (3840×2160) - 4K Ultra HD
- **1440p** (2560×1440) - 2K QHD
- **1080p** (1920×1080) - Full HD (default)
- **720p** (1280×720) - HD
- **480p** (854×480) - SD
- **360p** (640×360) - Low quality

### Audio Quality
- **128 kbps** - Default
- **192 kbps** - Recommended
- **256 kbps** - High quality
- **320 kbps** - Best quality

## Troubleshooting

### The application won't start
- Make sure you're using Windows 10 or later
- If SmartScreen blocks it: click "More info" → "Run anyway" (the app does not require administrator rights)
- Check if your antivirus is blocking the file

### Conversion fails
- Ensure your input file is a valid WebM file
- Check that you have enough disk space
- Make sure the output folder is writable

### File is too large
- Try using a lower quality setting (CRF 26)
- Reduce the resolution (e.g., 720p instead of 1080p)
- Lower the audio bitrate

## Tips

- The converted file will be saved in the same folder as your original file
- Higher quality settings result in larger file sizes
- Conversion time depends on video length and quality settings
- You can change the output file location by clicking the ✏️ button
